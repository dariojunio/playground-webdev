# Meu Playground WebDev

Bem-vindo(a) ao meu repositório de estudos! Aqui é o meu "playground" de desenvolvimento web, onde documento minha evolução, guardo trechos de código e subo os meus projetos práticos focados em **HTML5, CSS e JS**.

O objetivo deste espaço é fixar conceitos fundamentais da estrutura da web antes de avançar para lógica (JavaScript).

---

## 🛠️ Projetos Inclusos neste Repositório

Aqui estão os laboratórios práticos que desenvolvi para testar meus conhecimentos:

1.  **Tabela de Gatos e Cachorros:** Estudo focado em mesclagem de colunas com `colspan`, inserção de imagens e links funcionais internos às células da tabela.
2.  **Página de Biografia (About Kratos):** Exercício de listas ordenadas e aninhadas detalhando a franquia *God of War*, organizando os tópicos por eras (Grega e Nórdica).
3.  **Formulário de Cadastro:** Um formulário fictício aplicando diversos seletores, validação de e-mail nativa e caixas de seleção dinâmica.
